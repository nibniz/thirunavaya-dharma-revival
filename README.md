This repository contains the frontend for the Thirunavaya Mahamagha Mahotsavam / Dharma Revival site, built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

### Getting started

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

### Tech stack

- **Vite** (React + TypeScript)
- **Tailwind CSS**
- **shadcn-ui / Radix UI**

### Scripts

- **npm run dev**: Start the development server
- **npm run build**: Create a production build
- **npm run preview**: Preview the production build
- **npm run lint**: Run ESLint
